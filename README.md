# GACOR KANG (Beta)

Judi online page hunter on your server


### Installing

```
git clone https://github.com/unsa1nted/gacor-kang.git
```

### How to run

Run at once
```
node index.js
```

Run scheduled (every 5 minutes)
```
pm2 start index.js
```
